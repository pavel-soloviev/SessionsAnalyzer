# SessionsAnalyzer

Учебный проект для локального запуска задач на Apache Spark (Scala).

Программа обрабатывает события начала и конца сессии, извлекает параметры поиска через карточку и быстрый поиск, а также отслеживает открытия конкретных документов.

P.S. Вероятно не стоило так делать, но я закоммитил все логи сессий (10к файлов))) в директорию src/main/resources/sessions/.

## Результаты работы приложения

Программа успешно вычисляет два требуемых показателя:

### 1. Поиск документа ACC_45616 в карточке поиска
Подсчитано количество раз, когда в карточке производили поиск документа с идентификатором `ACC_45616` (именно в результатах поиска был этот документ т.е. шел в списке после CardSearchEnd).
* **Итоговое значение:** 479

### 2. Открытия документов из быстрого поиска
Подсчитано количество открытий каждого документа, найденного через быстрый поиск за каждый день. Результаты агрегированы и выводятся в консоль для **всех** доступных дат в хронологическом порядке.

**Формат вывода результатов в консоли:**
